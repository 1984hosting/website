    • Run Certbot: Just get a certificate

      {{#dns_plugins}} Run one of the commands in the “Examples” section of the instructions for your DNS provider. {{/dns_plugins}} {{^dns_plugins}} If you’re feeling more conservative and would like to make the changes to your Apache configuration by hand, run this command. 

      $ sudo {{base_command}} certonly --apache
      {{/dns_plugins}}

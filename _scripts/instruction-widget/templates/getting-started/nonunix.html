Certbot is currently only available for UNIX-like operating systems. Although EFF's Certbot might not work for your use case, there are many other clients written by other organizations and developers that you may be able to use to obtain a certificate from Let's Encrypt.

{{#advanced}}

Alternatively, you may be able to use Certbot if you have access to a UNIX-like machine. Using Certbot's manual plugin, you can obtain certificates without running the software on your target webserver. More information about this feature can be found in our documentation.

{{/advanced}}
